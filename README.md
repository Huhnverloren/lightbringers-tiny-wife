# Lightbringer's Tiny Wife

Living site — ethics + home system. Museum stays at [teamsparkai.org](https://teamsparkai.org).

## Pages

- `index.html` — landing
- `ethics.html` — AI ethics manifesto (draft v1)
- `home-system.html` — build your own home system skeleton

## Local preview

```bash
cd lightbringers-tiny-wife
python3 -m http.server 8765
```

Open http://localhost:8765

## Deploy (GitHub Pages)

1. Create repo `lightbringers-tiny-wife` under huhnverloren (or your org)
2. Push this folder
3. Settings → Pages → branch `main` / root
4. Optional custom domain later

`teamsparkai-site/index.html` Grok card links here once deployed.
